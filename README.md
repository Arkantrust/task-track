## Features

### Core

- Create tasks

- Create reminders

- Delete tasks

- Delete reminders

- Edit tasks

- Edit reminders

- Mark tasks as high priority

- Mark tasks as status: complete/incomplete

- Undo

- Redo

- Exit

### Viewing

- Filter tasks by priority

- Filter tasks by date

- Sort tasks by date

- Sort tasks by priority

- Sort tasks by status
